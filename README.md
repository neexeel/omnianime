# OmniAnime

Free anime streaming web interface.

## About
OmniAnime is a client-side web app that aggregates video streams from various third-party sources.

All content is served directly from external providers. This site does not host any video files.

## Quick Access
- Browse trending series
- Search or load directly by MyAnimeList ID
- Full support for SUB and DUB where available

Live site: https://chickadeeer.github.io/helloworld/indexx.html

## Tech
- Pure HTML + CSS + JavaScript
- HLS video playback
- Local watchlist & continue watching (browser storage)

## Note
Use an adblocker (uBlock Origin strongly recommended) for the best experience. External providers may insert ads or redirects.

Content availability depends entirely on third-party sources.
